- [vue-i18n](https://github.com/kazupon/vue-i18n)
- [portal-vue](https://github.com/LinusBorg/portal-vue) to prevent z-index bugs/hacks for modals and dialogs
- [tailwindcss](https://tailwindcss.com/docs)
